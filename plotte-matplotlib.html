Plotte med Matplotlib

Dette er den enkleste måten å lage en graf i Python med matplotlib: 

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [1, 4, 9, 16]

plt.plot(x, y)
plt.title("Min første graf")
plt.xlabel("x-verdi")
plt.ylabel("y-verdi")
plt.show()

Når du kjører dette, åpnes et vindu med en enkel graf. Du må ha matplotlib installert. Bruk eventuelt pip install matplotlib hvis du mangler det.
